## Backend Stack

### tRPC v11

End-to-end typesafe APIs without code generation.

**Package:** `@trpc/server` and `@trpc/react-query`

**Architecture:**

```typescript
// Server: apps/kitasuro/src/server/trpc/
import { initTRPC } from '@trpc/server'
import superjson from 'superjson'

const t = initTRPC.context<Context>().create({
  transformer: superjson,
})

export const appRouter = router({
  proposals: proposalsRouter,
  clients: clientsRouter,
  tours: toursRouter,
  // ... more routers
})

export type AppRouter = typeof appRouter
```

**Client:**
```typescript
// Client: fully typed, no code generation
import { trpc } from '@/lib/trpc'

const { data } = trpc.proposals.list.useQuery()
//    ^? Proposal[]
```

**Routers:**
- `clients` - Client management
- `proposals` - Proposal CRUD
- `tours` - Tour templates
- `activities` - Activity library
- `accommodations` - Accommodation database
- `nationalParks` - Park information
- `comments` - Proposal feedback
- `notes` - Internal notes
- `storage` - File uploads
- `contentLibrary` - Media management
- `settings` - Organization settings
- `onboarding` - User onboarding flows

**Features:**
- `protectedProcedure` - Authenticated endpoints
- `adminProcedure` - Admin-only endpoints
- SuperJSON transformer for Date, Map, Set serialization
- React Query integration
- Automatic error handling

### Drizzle ORM

TypeScript-first ORM with full type safety.

**Version:** 0.44.3

**Why Drizzle:**
- SQL-like syntax, close to raw SQL
- Excellent TypeScript inference
- Migration management with Drizzle Kit
- Zero runtime overhead
- PostgreSQL optimized

**Schema Definition:**
```typescript
// packages/db/src/schema.ts
import { pgTable, text, uuid, timestamp } from 'drizzle-orm/pg-core'

export const organizations = pgTable('organizations', {
  id: uuid('id').defaultRandom().primaryKey(),
  name: text('name').notNull(),
  slug: text('slug').notNull().unique(),
  planTier: PlanTier('plan_tier').default('free').notNull(),
  createdAt: timestamp('created_at').defaultNow().notNull(),
})

// Type inference
type Organization = typeof organizations.$inferSelect
type NewOrganization = typeof organizations.$inferInsert
```

**Queries:**
```typescript
import { db } from '@repo/db'
import { eq, and } from 'drizzle-orm'
import { proposals, proposalDays } from '@repo/db/schema'

// Type-safe queries with relations
const proposal = await db.query.proposals.findFirst({
  where: eq(proposals.id, proposalId),
  with: {
    days: {
      with: {
        activities: true,
        accommodations: true,
      },
    },
  },
})
```

**Migrations:**
```bash
cd packages/db
bun run generate  # Generate from schema changes
bun run migrate   # Apply migrations
bun run studio    # Visual database explorer
```

### PostgreSQL

Production-grade relational database.

**Client:** `postgres` (v3.4.7)

**Key Features:**
- JSONB columns for flexible data (activities, pricing)
- UUID primary keys
- Timestamp columns with timezone support
- Indexes for performance
- Foreign key constraints

**Connection:**
```typescript

const client = postgres(process.env.DATABASE_URL!)

```

## Authentication

### Better Auth

Modern authentication framework for Next.js.

**Version:** 1.4.18 (kitasuro), 1.4.2 (makisala)

**Features Implemented:**
- Email/password authentication
- OAuth (Google)
- Email verification
- Account linking
- Session management

**Configuration:**
```typescript
// apps/kitasuro/src/lib/auth.ts
import { betterAuth } from 'better-auth'
import { drizzleAdapter } from 'better-auth/adapters/drizzle'
import { organization } from 'better-auth/plugins'
import { passkey } from '@better-auth/passkey'

export const auth = betterAuth({
  database: drizzleAdapter(db, { provider: 'pg' }),
  emailAndPassword: {
    enabled: true,
    requireEmailVerification: true,
  },
  socialProviders: {
    google: {
      clientId: env.GOOGLE_CLIENT_ID,
      clientSecret: env.GOOGLE_CLIENT_SECRET,
    },
  },
  plugins: [
    passkey({ /* ... */ }),
    organization({ /* ... */ }),
  ],
})
```

**Client Usage:**
```typescript
import { authClient } from '@/lib/auth-client'

const { data: session } = authClient.useSession()
const { signIn, signOut, signUp } = authClient
```

**Version:** 7.60.0

```typescript

const schema = z.object({
  name: z.string().min(1),
  email: z.string().email(),
```
