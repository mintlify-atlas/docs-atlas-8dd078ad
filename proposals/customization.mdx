## Overview

Makisala provides multiple ways to customize proposal appearance:

- **Themes**: Pre-built design templates
- **Branding**: Organization logo and colors
- **Hero Images**: Custom header images
- **Day Images**: Preview images for each day

## Available Themes

Themes are selected in the Preview step of the proposal builder:

### Minimalistic Theme

```typescript
// components/themes/MinimalisticTheme.tsx
export default function MinimalisticTheme({ data }: { data: ItineraryData }) {
  return (
    <div className="min-h-screen bg-white">
      {/* Hero section */}
      <section className="relative h-[60vh]">
        {data.heroImage && (
          <img src={data.heroImage} className="h-full w-full object-cover" />
        )}
        <div className="absolute inset-0 bg-gradient-to-b from-transparent to-black/50" />
        <div className="absolute bottom-0 p-12 text-white">
          <h1 className="font-serif text-5xl">{data.tourTitle}</h1>
          <p className="mt-2 text-lg">{data.duration} • {data.startDate}</p>
        </div>
      </section>
      
      {/* Day-by-day itinerary */}
      <section className="mx-auto max-w-4xl px-8 py-16">
        {data.days.map((day) => (
          <DayCard key={day.dayNumber} day={day} />
        ))}
      </section>
    </div>
  );
}
```

**Characteristics:**
- Clean, modern design
- Elegant typography (serif headings)
- White background with accent colors
- Linear scroll layout
- Suitable for professional business proposals

### Kudu Theme

```typescript
// components/themes/kudu.tsx
export default function KuduTheme({ data }: { data: ItineraryData }) {
  return (
    <div className="snap-y snap-mandatory overflow-y-scroll">
      {/* Full-screen hero */}
      <section className="snap-start h-screen bg-black">
        <img src={data.heroImage} className="h-full w-full object-cover opacity-80" />
        <div className="absolute inset-0 flex items-center justify-center">
          <h1 className="font-serif text-7xl text-white">{data.tourTitle}</h1>
        </div>
      </section>
      
      {/* Snap-scroll days */}
      {data.days.map((day) => (
        <section key={day.dayNumber} className="snap-start h-screen">
          <FullscreenDayView day={day} />
        </section>
      ))}
    </div>
  );
}
```

**Characteristics:**
- Dark, immersive design
- Snap-scroll experience (full-screen sections)
- Large, dramatic imagery
- Minimalist text overlays
- Best for luxury, high-end experiences

## Theme Selection

Themes are selected in the preview page:

```typescript
// apps/kitasuro/src/app/itineraries/[id]/preview/page.tsx
const THEME_OPTIONS: { value: ThemeType; label: string; description: string }[] = [
  {
    value: 'minimalistic',
    label: 'Minimalistic',
    description: 'Clean, modern design with elegant typography',
  },
  { 
    value: 'kudu', 
    label: 'Kudu', 
    description: 'Dark, immersive snap-scroll theme' 
  },
  {
    value: 'discovery',
    label: 'Discovery',
    description: 'Immersive split-screen layout with full-size images',
  },
];

const {
  selectedTheme,
  setSelectedTheme
} = useBuilder();

// Theme picker popover
<Popover>
  <PopoverTrigger>
    <Button>
      <Palette className="h-4 w-4" />
      {THEME_OPTIONS.find(t => t.value === selectedTheme)?.label}
    </Button>
  </PopoverTrigger>
  <PopoverContent>
    {THEME_OPTIONS.map((theme) => (
      <button
        key={theme.value}
        onClick={() => setSelectedTheme(theme.value)}
        className={selectedTheme === theme.value ? 'bg-green-50' : ''}
      >
        {theme.label}
      </button>
    ))}
  </PopoverContent>
</Popover>
```

### Theme Access by Plan Tier

Theme availability is controlled by subscription plan:

```typescript
// lib/plans-config.ts
export const ALLOWED_THEMES_BY_TIER: Record<string, ThemeType[]> = {
  free: ['minimalistic'],
  starter: ['minimalistic', 'kudu'],
  pro: ['minimalistic', 'kudu', 'discovery'],
  business: ['minimalistic', 'kudu', 'discovery'],
};

// Theme validation on save
const plan = await getOrgPlan(ctx.orgId);
const allowedThemes = ALLOWED_THEMES_BY_TIER[plan.effectiveTier];
const validatedTheme = allowedThemes.includes(selectedTheme) 
  ? selectedTheme 
  : 'minimalistic';
```

## Hero Images

### Setting Hero Image

Hero images can be set from the preview page:

```typescript
const { heroImage, setHeroImage } = useBuilder();

// Open image picker
const [isHeroPickerOpen, setIsHeroPickerOpen] = useState(false);

<Dialog open={isHeroPickerOpen} onOpenChange={setIsHeroPickerOpen}>
  <DialogContent className="max-w-5xl">
    <ImagePickerContent
      onSelect={(imageUrl) => {
        setHeroImage(imageUrl);
        setIsHeroPickerOpen(false);
      }}
    />
  </DialogContent>
</Dialog>
```

### Image Picker Event

Themes can trigger the image picker:

```typescript
// From within a theme component
const handleOpenPicker = () => {
  window.dispatchEvent(new CustomEvent('openHeroImagePicker'));
};

<button onClick={handleOpenPicker}>
  Change Hero Image
</button>
```

## Custom CSS & Styling

### Theme-Specific Styles

Each theme has its own styling approach:

```typescript
// Minimalistic: Utility classes
<div className="mx-auto max-w-4xl px-8 py-16">

// Kudu: Snap scroll
<div className="snap-y snap-mandatory overflow-y-scroll h-screen">

// Discovery: Split screen
<div className="flex h-screen">
  <div className="w-1/2 overflow-y-auto" />
  <div className="sticky top-0 h-screen w-1/2" />
</div>
```

### Theme Variables

Themes use Tailwind utility classes with consistent color palette:

- **Primary**: `green-600`, `green-700`, `green-800`
- **Neutral**: `stone-50` through `stone-900`
- **Background**: `white`, `stone-50`, `black`
- **Accent**: `amber-500`, `amber-600`

## Best Practices

### Theme Selection

- **Minimalistic**: Professional business clients, multi-destination tours
- **Kudu**: Luxury lodges, high-end experiences, romantic getaways
- **Discovery**: Visual storytelling, wildlife photography tours, adventure travel

### Image Guidelines

- **Hero Images**: High-resolution (1920x1080+), landscape orientation
- **Day Images**: 1200x800+, representative of destination or accommodation
- **Logo**: SVG or PNG with transparent background, max 200px height

### Branding Consistency

- Set organization logo and description in settings
- Use consistent hero images across similar tour types
- Maintain color palette consistency with your brand
