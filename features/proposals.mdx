## Overview

Makisala's proposal management system allows you to transform tour templates into customized, client-ready proposals. Build detailed itineraries, manage pricing, and collaborate with your team—all in one place.

## Creating Proposals

Proposals are created from tour templates and can be fully customized for each client.

### Basic Information

<CodeGroup>
```typescript proposals.ts
// Creating a new proposal from a tour
const proposal = {
  name: "Tanzania Safari - Smith Family",
  tourId: "tour-uuid",
  clientId: "client-uuid",
  tourTitle: "7-Day Tanzania Safari Adventure",
  tourType: "Private Tour",
  status: "draft" // or "shared"
};
```
</CodeGroup>

Each proposal includes:
- **Name**: Internal reference for your team
- **Tour Title**: Client-facing title
- **Client**: Link to client record for contact management
- **Status**: Draft (internal) or Shared (sent to client)

## Pricing & Inclusions

### Flexible Pricing Structure

Configure pricing with multiple traveler types and extras:

```typescript schema.ts
pricingRows: [
  { id: "1", count: 2, type: "Adult", unitPrice: 5500 },
  { id: "2", count: 1, type: "Child", unitPrice: 3500 }
],
extras: [
  { id: "1", name: "Hot Air Balloon Safari", price: 550, selected: false },
  { id: "2", name: "Cultural Village Visit", price: 150, selected: true }
]
```

<Note>
  Use `hidePricing: true` to create proposals without showing costs to clients.
</Note>

### Inclusions & Exclusions

Clearly communicate what's included:

```typescript
inclusions: [
  "All park fees and government taxes",
  "Professional English-speaking guide",
  "4x4 safari vehicle with pop-up roof",
  "Accommodation as per itinerary"
],
exclusions: [
  "International flights",
  "Travel insurance",
  "Optional activities not mentioned",
  "Tips and gratuities"
]
```

## Sharing with Clients

Send proposals directly to clients via email.

```typescript proposals.ts:531-581
// Share proposal endpoint
await trpc.proposals.sendToClient({
  proposalId: "proposal-uuid",
  message: "Hello! Here's your customized safari proposal."
});
```

The system automatically generates:
- ✉️ Professional email with proposal details
- 🔗 Unique shareable link
- 📅 Start date and duration
- 💰 Pricing summary (if not hidden)

<Tip>
  Proposals can be viewed without authentication, making it easy for clients to access and share with travel companions.
</Tip>

## Client Acceptance

Clients can accept proposals directly from the shared link:

```typescript proposals.ts:583-643
// Client confirmation triggers email to agency
await trpc.proposals.confirm({
  proposalId: "proposal-uuid",
  clientName: "John Smith"
});
```

When a client accepts:
- 📧 Notification sent to agency's notification email
- 📊 Includes all proposal details and client information
- ⏰ Timestamp of acceptance

## Team Collaboration

### Assignments

Assign team members to proposals for better workflow management:

```typescript proposals.ts:482-529
// Assign a team member (admin only)
await trpc.proposals.assign({
  proposalId: "proposal-uuid",
  userId: "team-member-uuid"
});

// Remove assignment
await trpc.proposals.unassign({
  proposalId: "proposal-uuid",
  userId: "team-member-uuid"
});
```

<Info>
  The proposal creator is automatically assigned when saving for the first time. Additional assignments require admin privileges.
</Info>

### Dashboard Filtering

Team members can filter proposals:
- **My Proposals**: Only assigned to you
- **All Proposals**: Everything in your organization

```typescript proposals.ts:84-127
await trpc.proposals.listForDashboard({ filter: "mine" });
```
